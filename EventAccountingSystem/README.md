# Система учета мероприятий

## Описание проекта
Приложение для учета мероприятий, разработанное на C# с использованием ASP.NET Core Web API и Windows Forms. Позволяет создавать, просматривать, редактировать и удалять мероприятия, а также отслеживать их статус.

## Технологии
- C# .NET 8
- ASP.NET Core Web API
- Windows Forms
- System.Text.Json
- REST API
- JSON файловое хранилище

## Функциональные возможности
- Просмотр списка всех мероприятий
- Добавление нового мероприятия
- Редактирование существующего мероприятия
- Удаление мероприятия
- Изменение статуса мероприятия
- Сохранение данных в JSON файл

## Структура проекта
- EventAccountingSystemAPI - серверная часть (REST API)
- EventAccountingSystemUI - клиентская часть (Windows Forms)

## Запуск приложения
1. Скачать ZIP-архив
2. Разархивировать ZIP-архив в любое место на компьютере
3. Нажать правой кнопкой мыши по папке проекта и выбрать запусти с помощью Visual Studio
4. Разверните папку EventAccountingSystem и дважды кликните по EventAccountingSystem.sln
5. Перед запуском убедитесь что у вас запустится одновременно оба проекта (Правой кнопкой мыши по «Решению» - Настройка начальных проектов – убедитесь что галочка на «Несколько запускаемых проектов» и в обоих ячейках действия стоит «Запустить»).
6. Нажмите F5 и протестируйте приложение.


## API Endpoints
| Метод  | URL                     | Назначение                 |
|--------|-------------------------|----------------------------|
| GET    | /api/events             | получить все мероприятия   |
| GET    | /api/events/{id}        | получить мероприятие по ID |
| POST   | /api/events             | создать мероприятие        |
| PUT    | /api/events/{id}        | обновить мероприятие       |
| PUT    | /api/events/{id}/status | изменить статус            |
| DELETE | /api/events/{id}        | удалить мероприятие        |
